What changed

Header rewrite.
My Places · Ask from memory
Your places · Everywhere you’ve been worth remembering
Search demoted.
Smaller pill, lighter background, sits below header. Placeholder is plain (“Search places, dishes, cities”) — no longer framed as “ask” / chatbot voice.
Lying chip row removed.
The first-chip-always-active styling was misleading. Replaced with three clearly differentiated sections instead of three surfaces that do the same job.
New: “Your most-loved”.
Horizontal strip of 2–3 cards. Source from existing signals: userIsGoTo, userLoved, repeat visitCount. Sand badge tells you why this card is here. Answers “which places matter to me?” before you have to scroll.
By city: count + cuisine sub.
Replaces “Best in SF?” query cards. Same destination, clearer affordance — you see the count, you see what kind of food lives there, you tap to drill in.
Recently logged.
Replaces “Quick access”. Same data, more honest label. If a user just added a place, this is where they verify it landed.
Color discipline.
Sand for people-shaped signals (badges, city counts), text white/muted otherwise. No orange on this page — orange is the recommendation voice and lives on Home.
What I dropped.
“You might ask” suggested-queries section. The functionality still exists via search — typing “best in” triggers the same intent — but it stops being a separate surface competing for attention.
